# Show All User Detail

Show all user's id and full name

**URL** : `/api/v1/spiritanime`

**Method** : `GET`

**Auth required** : NO

**Permissions required** : None

**Data**: `{}`

## Success Response


**Code** : `200 OK`

**Content example**

```json
{
    "users": [
        {
            "user_id": 170256,
            "full_name": "212312111"
        },
        {
            "user_id": 154997,
            "full_name": "test"
        }
    ]
}
```
